# 💰 EMI Calculator

A responsive **EMI (Equated Monthly Installment) Calculator** built using **HTML, CSS, Bootstrap, and JavaScript**. 
This application helps users calculate their monthly loan EMI, total interest payable, and total payment based on the loan amount, interest rate, and loan tenure.

## 🚀 Live Demo
https://sagar-chand.github.io/EMI-calulator/

## ✨ Features

* 📌 Calculate monthly EMI instantly
* 💰 Calculate total interest payable
* 📊 Display total repayment amount
* 📱 Fully responsive design
* 🎨 Clean and user-friendly interface
* ⚡ Fast and lightweight

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5
* JQuery 
* JavaScript (ES6)

## 📖 How to Use

1. Enter the **Loan Amount**
2. Enter the **Annual Interest Rate**
3. Enter the **Loan Tenure** (Months/Years)
4. Click the **Calculate** button
5. View:

   * Monthly EMI
   * Total Interest
   * Total Amount Payable

## 📷 Formula Used

```
EMI = [P × R × (1 + R)^N] / [(1 + R)^N − 1]
```

Where:

* **P** = Principal Loan Amount
* **R** = Monthly Interest Rate
* **N** = Loan Tenure (Months)
